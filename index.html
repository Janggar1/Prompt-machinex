Image to Prompt Machine · Vercel Ready

Upload gambar, lalu ubah jadi prompt image yang rapi.

Tool ini fokus untuk image to prompt. Kamu cukup upload gambar, pilih target model, lalu AI akan menganalisis subjek, outfit, pose, komposisi, kamera, lighting, background, style, serta menghasilkan prompt siap pakai.

Input Gambar

Upload 1 gambar. Catatan tambahan opsional kalau kamu mau prompt diarahkan ke model tertentu.
Readiness score 0%
Skor ini cuma menunjukkan kesiapan input image-to-prompt, bukan jaminan akurasi mutlak.
Simpan API key di Vercel Environment Variables dengan nama OPENAI_API_KEY. Tidak perlu ditaruh di HTML.

Output

Hasil bisa langsung dicopy ke generator image.
Output prompt akan muncul di sini setelah kamu upload gambar lalu klik "Generate Image Prompt".
Catatan: prompt hasil AI tetap bisa beda tipis antar model image generator. Untuk hasil paling mirip, gunakan model yang sama, seed tetap, dan reference image bila platform mendukung.
